## Objectif du projet

L'objectif de ce projet est d'appliquer les techniques de la
recherche opérationnelle pour résoudre trois problèmes
d'optimisation issus de domaines différents :

- Optimisation combinatoire
- Logistique
- Application réelle en Mauritanie
